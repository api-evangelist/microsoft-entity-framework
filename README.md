# Microsoft Entity Framework (microsoft-entity-framework)
Microsoft Entity Framework is an object-relational mapping (ORM) framework  for .NET developers. It eliminates the need for most data-access code by  enabling developers to work with databases using .NET objects.


**URL:** [Visit APIs.json URL](https://docs.microsoft.com/en-us/ef/)
